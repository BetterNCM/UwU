<h1 align="center">BetterUniverse</h1>
<h3 align="center">for QQNT/Electron</h3>
<h4 align="center">Electron 客户端插件管理器</h4>

由于可预期的 QQNT 安全性更新，此项目将在第一个 QQNT For Windows 正式版（无白名单版本）推出后开源。

目前此仓库仅作文档及版本发布用。

本插件仅为优化 QQ 和其他 Electron 应用的使用体验而生，如有侵权，请联系我们删除。

# 注意
本插件不保证在所有版本上的通用可用性，也不保证安全性（封号，崩溃等）。

在使用此插件后所遇到的 Bug，崩溃等，请不要向官方反馈，而应在此 Repo 下发送 Issue。

本插件可能将于未来的某一天突然跑路。

# 功能
  - Koishi 机器人
  - 主题
  - RedProtocol 机器人协议
  - 自动执行 Pangu.js
  - 加密通话（假）【发送 enc#+文本】
  - 防撤回
  - 直接打开小程序对应网页（部分兼容）
  - 自动分类下载的群文件
  - WIP……

  画个饼：
  - 消息一键跳转（到上条未读，在 at 间切换等等）
  - 更多的主题
  - +1 按钮
